Inicio Características Ventajas y Desventajas

PHP

Ventajas

  • Es una plataforma independiente. Las aplicaciones basadas en PHP pueden ejecutarse en cualquier sistema operativo como UNIX, Linux y Windows, etc.
  • Se pueden cargar fácilmente aplicaciones basadas en PHP y conectadas a la base de datos.
  • Ayuda a reutilizar un código equivalente y no tiene que escribir un código largo y una estructura sofisticada para el evento de las aplicaciones web.
  • Ayuda a administrar el código fácilmente.
  • Tiene un potente soporte de biblioteca para utilizar varios módulos de funciones para la representación de datos.

    • Desventajas

  • No es adecuado para aplicaciones web gigantes basadas en contenido.
  • Los frameworks PHP aprendieron a usar las funcionalidades integradas de PHP para evitar escribir código adicional.
  • El uso de más funciones del marco y las herramientas de PHP provoca un rendimiento deficiente de las aplicaciones en línea.
  • PHP no permite cambios o modificaciones en el comportamiento principal de las aplicaciones en línea.
  • Los frameworks PHP no son equivalentes en comportamiento, también lo son su rendimiento y características.
